Smart Home Simulator Wapp

Background task of this wapp creates a Network under your user account. Therefore, a popup should show asking for permission to save data. You need to accept it in order to have this application working. In case a popup does not show, you can view it in Wappsto in the "Notifications" panel.

This network will generate 1 Device with 3 Values: humidity, temperature, and CO2. All your user data can be viewed using "My Data" wapp which should be pre-installed by default.

The background task is configured to save "new" data every minute, which means that every State of the value will save historical data from the moment you have installed this wapp.
