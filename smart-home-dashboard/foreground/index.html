Smart Home Dashboard Wapp

This application is made as an example of how to require a Network with the name "Smart Home Simulator".

The required Network is created by the wapp "Smart Home Simulator", therefore, you need to have it installed for this wapp to work. A permission request made by "main.js" will trigger a popup that asks for premission to access a network from your list of networks that matches the above name criteria. You need to select one and allow to use it. In case a popup does not show, you can view it in Wappsto in the "Notifications" panel.
